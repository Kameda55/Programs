# TAKIGYO

## 概要

Visual Basicを用いて制作した落ちてくる岩を避けるゲームです。
シンプルながら背景や演出にこだわるように手を加えました。

## 開発背景

授業の最終課題として制作しました。

## 開発環境について

Visual Studioを使い作成しました。

## 開発期間

おおよそ4週間ほどで完成しました。

## 操作方法

- マウス移動：キャラの左右移動

## 使用技術

- Visual Basic

## 工夫した点

- ゲーム性はシンプルに、でも味気なくならないように背景などにこだわった
- カーソルがウインドウ外に出ても自分のキャラが画面外に行かないように工夫
- 授業評価として優評価を頂けた
